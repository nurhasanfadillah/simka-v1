<objective>
## Goal
Bangun backend endpoint Users CRUD (list/create/update/deactivate) dan PATCH /auth/change-password di dalam auth module yang sudah ada.

## Purpose
Super Admin butuh cara menambah operator/bendahara tanpa akses langsung ke DB. Change password memungkinkan operator mengganti password default mereka.

## Output
- GET /users — list semua user + role name
- POST /users — create user baru dengan role
- PATCH /users/:id — update nama/email/roleId/isActive
- PATCH /auth/change-password — user ganti password sendiri (butuh currentPassword)
</objective>

<acceptance_criteria>

## AC-1: List Users
```gherkin
Given Super Admin melakukan GET /users dengan Bearer token
When request diterima
Then response array users dengan field: id, name, email, roleName, isActive, lastLogin, createdAt
And dilindungi @RequirePermissions('user.view')
```

## AC-2: Create User
```gherkin
Given Super Admin POST /users dengan { name, email, password, roleId }
When request valid
Then user baru tersimpan dengan password di-hash bcrypt
And response user yang baru dibuat (tanpa password field)
And dilindungi @RequirePermissions('user.manage')
```

## AC-3: Update User
```gherkin
Given Super Admin PATCH /users/:id dengan partial { name?, email?, roleId?, isActive? }
When request valid
Then field yang dikirim terupdate, field lain tidak berubah
And response user terupdate (tanpa password)
And dilindungi @RequirePermissions('user.manage')
```

## AC-4: Change Password
```gherkin
Given user login melakukan PATCH /auth/change-password dengan { currentPassword, newPassword }
When currentPassword cocok dengan hash di DB
Then password terupdate dengan hash baru
And response { message: 'Password berhasil diubah' }
And jika currentPassword salah, return 400
```

       ```ts
       import { IsEmail, IsNotEmpty, IsString, IsNumber, IsOptional, IsBoolean, MinLength } from 'class-validator'

       export class CreateUserDto {
         @IsNotEmpty() @IsString() name: string
         @IsEmail() email: string
         @IsString() @MinLength(6) password: string
         @IsNumber() roleId: number
       }

       export class UpdateUserDto {
         @IsOptional() @IsString() name?: string
         @IsOptional() @IsEmail() email?: string
         @IsOptional() @IsNumber() roleId?: number
         @IsOptional() @IsBoolean() isActive?: boolean
       }

       export class ChangePasswordDto {
         @IsString() @IsNotEmpty() currentPassword: string
         @IsString() @MinLength(6) newPassword: string
       }
       ```

    2. Buat `users.service.ts` — @Injectable(), inject DRIZZLE:
       - `findAll()`: SELECT id, name, email, isActive, lastLogin, createdAt + JOIN roles → roleName. Return array
       - `create(dto: CreateUserDto)`: cek email unique (throw ConflictException jika ada), hash password bcrypt(10), INSERT, return user tanpa password
       - `update(id: number, dto: UpdateUserDto)`: cek user exists (throw NotFoundException), UPDATE hanya field yang ada di dto, return updated user tanpa password
       - `changePassword(userId: number, dto: ChangePasswordDto)`: fetch user by id, bcrypt.compare(currentPassword, user.password), jika gagal throw BadRequestException('Password saat ini tidak cocok'), hash newPassword, UPDATE

    Avoid: jangan expose password field di response apapun.
    Avoid: jangan buat module baru — inject ke AuthModule yang ada.

    1. Buat `users.controller.ts`:
       ```ts
       @Controller('users')
       @UseGuards(JwtAuthGuard, RbacGuard)
       export class UsersController {
         constructor(private usersService: UsersService) {}

         @Get()
         @RequirePermissions('user.view')
         findAll() { return this.usersService.findAll() }

         @Post()
         @RequirePermissions('user.manage')
         create(@Body() dto: CreateUserDto) { return this.usersService.create(dto) }

         @Patch(':id')
         @RequirePermissions('user.manage')
         update(@Param('id', ParseIntPipe) id: number, @Body() dto: UpdateUserDto) {
           return this.usersService.update(id, dto)
         }
       }
       ```
       Import: JwtAuthGuard dari guards/, RbacGuard, RequirePermissions dari decorators/

    2. Di `auth.controller.ts`, tambah endpoint setelah @Get('me'):
       ```ts
       @Patch('change-password')
       async changePassword(
         @CurrentUser() user: { id: number },
         @Body() dto: ChangePasswordDto,
       ) {
         return this.usersService.changePassword(user.id, dto)
       }
       ```
       Inject UsersService ke AuthController constructor.
