<objective>
## Goal
Tambah field `isActive` ke tabel students (backend + migration), endpoint toggle-active, dan update frontend siswa/index.tsx dengan action menu ⋮ (DropdownMenu) yang menggantikan tombol teks, plus badge status aktif/non-aktif.

## Purpose
Halaman Data Siswa saat ini punya tombol Ubah/Hapus yang menyebabkan baris tabel overflow di layar kecil. Action icon menu menyelesaikan overflow sekaligus menambah fitur kelola aktif/non-aktif siswa.

## Output
- `isActive` boolean di tabel students (Drizzle schema + migration)
- PATCH `/master/students/:id/toggle-active` endpoint
- Student type diupdate dengan `isActive`
- Tabel siswa pakai DropdownMenu ⋮ dengan 4 aksi: Edit / Toggle Aktif / Kelas / Hapus
- Badge "Aktif" (hijau) / "Non-aktif" (abu) di kolom status
</objective>

    **Jalankan migration:**

    **students.service.ts — update findAll dan findOne agar include isActive:**
    Field ini otomatis ter-include karena Drizzle select * dari schema — pastikan response shape tidak strip field ini.

    Tambahkan method baru `toggleActive(id: number)`:
    ```typescript
    async toggleActive(id: number) {
      const student = await this.findOne(id)
      const updated = await this.db
        .update(studentsTable)
        .set({ isActive: !student.isActive, updatedAt: new Date() })
        .where(eq(studentsTable.id, id))
        .returning()
      return updated[0]
    }
    ```

    **students.controller.ts — tambahkan endpoint:**
    Letakkan SEBELUM `@Patch(':id')` (literal route harus mendahului param route):
    ```typescript
    @Patch(':id/toggle-active')
    @HttpCode(HttpStatus.OK)
    @RequirePermissions('student.manage')
    toggleActive(@Param('id', ParseIntPipe) id: number) {
      return this.studentsService.toggleActive(id)
    }
    ```

    **update-student.dto.ts — tambahkan isActive optional:**
    ```typescript
    @IsBoolean()
    @IsOptional()
    isActive?: boolean
    ```

    **siswa/index.tsx — perubahan UI:**

    1. **Import tambahan:**
    ```typescript

    import {
      DropdownMenu,
      DropdownMenuContent,
      DropdownMenuItem,
      DropdownMenuTrigger,
      DropdownMenuSeparator,
    } from '@/components/ui/dropdown-menu'
    ```

    2. **State tambahan:**
    ```typescript

    ```

    3. **Handler toggle:**
    ```typescript
    const handleToggleActive = async (student: Student) => {
      setTogglingId(student.id)
      try {
        await apiClient.patch(`/master/students/${student.id}/toggle-active`)
        await fetchData()
      } catch {
        setError('Gagal mengubah status siswa.')
      } finally {
        setTogglingId(null)
      }
    }
    ```

    4. **Kolom Status di tabel — tambahkan badge isActive:**
    Setelah badge `studentStatus` yang sudah ada, tambahkan badge `isActive`:
    ```tsx
    <span className={`text-xs px-1.5 py-0.5 rounded font-medium ml-1 ${
      s.isActive
        ? 'bg-green-100 text-green-700'
        : 'bg-gray-100 text-gray-500'
    }`}>
      {s.isActive ? 'Aktif' : 'Non-aktif'}
    </span>
    ```

    5. **Ganti tombol Ubah/Hapus dengan DropdownMenu:**
    Hapus tombol `<Button>Ubah</Button>` dan `<Button>Hapus</Button>` di setiap baris.
    Ganti dengan:
    ```tsx
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="ghost" size="sm" className="h-7 w-7 p-0">
          <MoreVertical className="w-4 h-4" />
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent align="end">
        <DropdownMenuItem onClick={() => handleEdit(s)}>
          Edit
        </DropdownMenuItem>
        <DropdownMenuItem
          onClick={() => handleToggleActive(s)}
          disabled={togglingId === s.id}
        >
          {s.isActive ? 'Nonaktifkan' : 'Aktifkan'}
        </DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem
          className="text-red-600 focus:text-red-600"
          onClick={() => handleDelete(s)}
        >
          Hapus
        </DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
    ```
    Catatan: Cek apakah `handleEdit` dan `handleDelete` sudah ada di file, sesuaikan nama fungsi dengan yang existing.
    Untuk tombol "Kelas" (akses mapping kelas) bisa ditambahkan sebagai DropdownMenuItem yang navigate ke `/master/kelas/mapping`.
