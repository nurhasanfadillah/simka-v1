- /pengaturan/roles: tabel roles + create modal + assign permissions modal (checkbox list)
- /pengaturan/profil: tambah section form ganti password (PATCH /auth/change-password)

<acceptance_criteria>

## AC-1: Tabel Roles
```gherkin
Given user membuka /pengaturan/roles
When halaman dimuat
Then tampil tabel dengan kolom: Nama Role, Jumlah Permission, Aksi
And data di-fetch dari GET /roles
And loading skeleton ditampilkan saat fetch
```

## AC-2: Create Role
```gherkin
Given user klik "Tambah Role"
When modal create terbuka dan name diisi
Then POST /roles dikirim
And modal tertutup dan tabel refresh
And jika 409, tampil error "Nama role sudah dipakai"
```

## AC-3: Assign Permissions
```gherkin
Given user klik "Atur Permission" pada baris role
When modal assign terbuka
Then tampil checkbox list semua permissions yang tersedia (dikumpulkan dari GET /roles)
And permission yang sudah dimiliki role = checked
When user checklist/uncheck dan klik Simpan
Then PATCH /roles/:id/permissions dengan { permissionIds: [...] } dikirim
And modal tertutup dan tabel refresh
```

## AC-4: Form Ganti Password
```gherkin
Given user membuka /pengaturan/profil
When scroll ke bawah section profil
Then tampil form "Ganti Password" dengan field: currentPassword, newPassword (min 6), confirmPassword
When form di-submit dengan currentPassword benar dan newPassword match
Then PATCH /auth/change-password dikirim
And tampil pesan sukses "Password berhasil diubah"
And jika currentPassword salah (400), tampil error dari server
And jika newPassword ≠ confirmPassword, tampil error validasi
```

    2. Replace seluruh isi `/pengaturan/roles/index.tsx` dengan halaman Roles Management:

       State: roles (RoleItem[]), loading, error, saving
       State modal: createOpen (boolean), assignOpen (boolean)
       State assign: selectedRole (RoleItem|null), checkedIds (Set<number>)
       State allPermissions (PermissionItem[]) — dikumpulkan dari GET /roles response

       Zod schema create: { name: z.string().min(1, 'Wajib diisi') }

       fetchRoles():
         GET /roles → setRoles(res.data)
         Kumpulkan allPermissions: dari res.data, flatMap permissions, dedup by id:
         ```ts
         const permMap = new Map<number, PermissionItem>()
         res.data.forEach(r => r.permissions.forEach(p => permMap.set(p.id, p)))
         setAllPermissions(Array.from(permMap.values()).sort((a, b) => a.code.localeCompare(b.code)))
         ```

       openCreate(): reset form, setCreateOpen(true)

       onSubmitCreate(formData):
         POST /roles { name }
         catch 409 → setError('Nama role sudah dipakai')
         success → setCreateOpen(false), fetchRoles()

       openAssign(role: RoleItem):
         setSelectedRole(role)
         setCheckedIds(new Set(role.permissions.map(p => p.id)))
         setAssignOpen(true)

       onSubmitAssign():
         PATCH /roles/:selectedRole.id/permissions { permissionIds: Array.from(checkedIds) }
         success → setAssignOpen(false), fetchRoles()

       Tabel kolom: Nama Role, Jumlah Permission (badge), Aksi
       - Jumlah Permission: `<span className="inline-flex px-2 py-1 bg-blue-50 text-blue-700 rounded text-xs">{role.permissions.length} permission</span>`
       - Aksi: tombol "Atur Permission" (Settings icon atau ShieldCheck)

       Modal Create: field Nama Role saja, tombol Simpan

       Modal Assign Permissions:
         Title: "Atur Permission — {selectedRole.name}"
         Scrollable list (max-h-72 overflow-y-auto) dari allPermissions:
         ```tsx
         {allPermissions.map(p => (
           <label key={p.id} className="flex items-center gap-3 py-2 px-3 hover:bg-gray-50 rounded cursor-pointer">
             <input
               type="checkbox"
               checked={checkedIds.has(p.id)}
               onChange={e => {
                 const next = new Set(checkedIds)
                 e.target.checked ? next.add(p.id) : next.delete(p.id)
                 setCheckedIds(next)
               }}
               className="w-4 h-4 accent-[#00A651]"
             />
             <div>
               <span className="text-sm font-medium text-gray-800">{p.name}</span>
               <span className="ml-2 text-xs font-mono text-gray-400">{p.code}</span>
             </div>
           </label>
         ))}
         ```
         Footer: tombol Batal + Simpan (bg-[#00A651])

       Header page: "Role & Akses" / "Manajemen role dan hak akses pengguna"
       Avoid: jangan gunakan Tanstack Table — pakai HTML table seperti kelas/index.tsx

    ```ts
    const pwSchema = z.object({
      currentPassword: z.string().min(1, 'Wajib diisi'),
      newPassword: z.string().min(6, 'Minimal 6 karakter'),
      confirmPassword: z.string().min(1, 'Wajib diisi'),
    }).refine(d => d.newPassword === d.confirmPassword, {

      path: ['confirmPassword'],
    })

    ```

    useForm dengan pwSchema di dalam komponen (terpisah dari form utama jika ada).

    onSubmitPw(data: PwFormData):

      catch AxiosError status 400 → setPwError(err.response?.data?.message ?? 'Password saat ini tidak cocok')
