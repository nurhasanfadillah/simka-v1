## Goal
Bangun halaman Laporan Tunggakan (`/laporan/tunggakan`) dan Rekap POS (`/laporan/rekap-pos`) dengan filter, tabel data, summary card, dan tombol export Excel/PDF.

## Purpose
Bendahara butuh laporan tunggakan siswa (filter kelas/unit/tahun ajaran) dan rekap penerimaan per POS (filter tanggal/tahun ajaran) untuk audit keuangan sekolah.

## Output
- 2 halaman laporan baru terdaftar di router
- Types untuk response kedua endpoint
- Consistent dengan pola 08-01 (harian/bulanan/tahunan)
</objective>

