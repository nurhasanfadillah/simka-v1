## Backend API (existing, ready)
- `GET /api/reports/harian?date=YYYY-MM-DD` → `{ date, totalPenerimaan, jumlahTransaksi, transactions[] }`
- `GET /api/reports/harian/export/pdf?date=` → binary PDF
- `GET /api/reports/harian/export/excel?date=` → binary Excel
- `GET /api/reports/bulanan?month=N&year=N&classId&unitId` → `{ month, year, totalPenerimaan, jumlahTransaksi, perHari[] }`
- `GET /api/reports/bulanan/export/pdf?month&year` → binary PDF
- `GET /api/reports/bulanan/export/excel?month&year` → binary Excel
- `GET /api/reports/tahunan?schoolYearId=N` → `{ schoolYearId, totalPenerimaan, jumlahTransaksi, perBulan[] }`
- `GET /api/master/school-years` — dropdown tahun ajaran
- `GET /api/master/classes` — optional filter

## AC-1: Halaman Laporan Harian
```gherkin
Given user membuka /laporan/harian
When user memilih tanggal (date picker, default today)
Then tabel tampil: No. Transaksi, Siswa, NIS, Nominal, Status, Tanggal
  Dan summary card: Total Penerimaan (Rupiah) + Jumlah Transaksi
  Dan tombol "Export Excel" dan "Export PDF" di header
  Dan klik export → download file binary
```

## AC-2: Halaman Laporan Bulanan
```gherkin
Given user membuka /laporan/bulanan
When user memilih bulan + tahun (2 select)
Then tampil summary: Total Penerimaan + Jumlah Transaksi
  Dan tabel per hari: Tanggal, Penerimaan, Jumlah Transaksi
  Dan tombol export Excel/PDF
```

## AC-3: Halaman Laporan Tahunan
```gherkin
Given user membuka /laporan/tahunan
When user memilih tahun ajaran (dropdown, REQUIRED)
Then tampil summary: Total Penerimaan + Jumlah Transaksi
  Dan tabel per bulan: Bulan, Penerimaan, Jumlah Transaksi
  Dan tombol export Excel/PDF
```

## AC-4: Loading & Error
```gherkin
Given user di halaman laporan
When data sedang di-fetch
Then skeleton loading tampil
  Dan jika API error, banner merah tampil
  Dan untuk laporan tahunan: jika tahun ajaran belum dipilih, tampil "Pilih tahun ajaran untuk melihat laporan"
```
