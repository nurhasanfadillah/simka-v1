<task type="auto">
  <name>Task 2: Service voidTransaction() — reverse bill status dalam DB transaction</name>
  <files>apps/backend/src/transactions/transactions.service.ts</files>
  <action>
    Tambah method `voidTransaction(id, voidedById, voidReason)` ke TransactionsService.

    **Import tambahan yang dibutuhkan:** `BadRequestException` dari `@nestjs/common`.

    **Logic method:**

    Semua dalam `this.db.transaction(async (tx) => { ... })`:

    **1. Load transaction + items dalam satu tx:**
    ```
    const [trx] = await tx.select().from(transactions).where(eq(transactions.id, id)).limit(1);
    if (!trx) throw new NotFoundException('Transaksi tidak ditemukan');
    if (trx.status === 'void') throw new BadRequestException('Transaksi sudah di-void');

    const items = await tx.select().from(transactionItems).where(eq(transactionItems.transactionId, id));
    ```

    **2. Update transaction → void:**
    ```
    await tx.update(transactions).set({
      status: 'void',
      voidedAt: new Date(),
      voidedBy: voidedById,
      voidReason,
      updatedAt: new Date(),
    }).where(eq(transactions.id, id));
    ```

    **3. Reverse bill_months (hanya yang punya billMonthId):**
    ```
    const billMonthIds = items
      .map(i => i.billMonthId)
      .filter((id): id is number => id != null);

    for (const bmId of billMonthIds) {
      await tx.update(billMonths)
        .set({ status: 'belum_bayar', paidAt: null })
        .where(eq(billMonths.id, bmId));
    }
    ```

    **4. Recalculate bills.status (logika identik dengan create, tapi setelah reversal):**
    ```
    const uniqueBillIds = [...new Set(items.map(i => i.billId))];
    for (const billId of uniqueBillIds) {
      const months = await tx.select({ status: billMonths.status })
        .from(billMonths).where(eq(billMonths.billId, billId));

      let newBillStatus: 'belum_bayar' | 'cicilan' | 'lunas';
      if (months.length === 0) {
        newBillStatus = 'belum_bayar';  // void context: 0 bulan = belum bayar
      } else {
        const lunasCount = months.filter(m => m.status === 'lunas').length;
        if (lunasCount === months.length) newBillStatus = 'lunas';
        else if (lunasCount > 0) newBillStatus = 'cicilan';
        else newBillStatus = 'belum_bayar';
      }

      await tx.update(bills)
        .set({ status: newBillStatus, updatedAt: new Date() })
        .where(eq(bills.id, billId));
    }
    ```

    **5. Return:**
    ```
    return {
      id: trx.id,
      transactionNumber: trx.transactionNumber,
      status: 'void' as const,
      voidedAt: new Date(),
      voidReason,
    };
    ```

    PENTING: Pada void context, bills dengan 0 bill_months → 'belum_bayar' (kebalikan dari create yang pakai 'lunas').
    AVOID: Jangan ubah method lain (create, findAll, findOne).
