<objective>
## Goal
Sidebar SIMKA bisa di-collapse dengan transisi animasi halus, gradasi background lebih dramatis, dan active state indicator yang lebih modern.

## Purpose
Saat ini sidebar fixed 264px tanpa collapse. Untuk layar kecil dan preferensi user, sidebar bisa dikecilkan ke icon-only (~64px). Gradasi background saat ini flat `from-primary to-primary-light` — perlu dibuat lebih dramatis dengan mid-stop color.

## Output
1. Sidebar collapsible dengan animasi width transition (264px ↔ 64px)
2. Toggle button (chevron) di area logo
3. Icon-only mode: nav items hanya icon + tooltip, section label hidden, user info hidden, logo text hidden
4. Enhanced gradient: `from-primary via-primary to-primary-light` atau 3-stop untuk depth lebih
5. Active state: tambah left border accent + background subtle highlight (bukan full green)
6. AppLayout adjusted — main content margin mengikuti sidebar width
</objective>
