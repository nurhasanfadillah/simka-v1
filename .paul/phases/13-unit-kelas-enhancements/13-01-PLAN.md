<objective>
## Goal
Tambah DELETE endpoint untuk School Units dan Classes, masing-masing dengan guard yang mencegah hapus data yang masih dipakai.

## Purpose
Backend perlu mendukung penghapusan unit sekolah dan kelas yang salah input, dengan proteksi agar tidak merusak integritas data relasional.

## Output
- `DELETE /master/school-units/:id` — guard: tolak jika ada kelas terkait
- `DELETE /master/classes/:id` — guard: tolak jika ada studentClasses atau paymentTemplates terkait
</objective>

## AC-1: Hapus unit sekolah tanpa kelas terkait
```gherkin
Given unit sekolah tidak memiliki kelas terkait
When DELETE /master/school-units/:id dipanggil
Then unit sekolah terhapus dari database
  AND response 204 No Content
```

## AC-2: Tolak hapus unit sekolah jika ada kelas
```gherkin
Given unit sekolah masih memiliki kelas terkait
When DELETE /master/school-units/:id dipanggil
Then backend return 400 BadRequestException
  AND pesan: 'Unit sekolah tidak dapat dihapus karena masih memiliki kelas terkait'
```

## AC-3: Hapus kelas tanpa data terkait
```gherkin
Given kelas tidak memiliki studentClasses maupun paymentTemplates terkait
When DELETE /master/classes/:id dipanggil
Then kelas terhapus dari database
  AND response 204 No Content
```

## AC-4: Tolak hapus kelas jika ada data terkait
```gherkin
Given kelas memiliki studentClasses atau paymentTemplates terkait
When DELETE /master/classes/:id dipanggil
Then backend return 400 BadRequestException
  AND pesan: 'Kelas tidak dapat dihapus karena sudah memiliki data terkait'
```

  <files>

  <action>
    Di `school-units.service.ts`:
    1. Tambah import: `count` dari drizzle-orm, `classes` dari `../../db/schema`
    2. Tambah method `remove(id: number)`:
       - Panggil `findOne(id)` — throw NotFoundException jika tidak ada
       - Query count: `this.db.select({ value: count() }).from(classes).where(eq(classes.schoolUnitId, id))`
       - Jika count > 0 → throw `BadRequestException('Unit sekolah tidak dapat dihapus karena masih memiliki kelas terkait')`
       - Eksekusi `this.db.delete(schoolUnits).where(eq(schoolUnits.id, id))`

    Di `school-units.controller.ts`:
    1. Tambah import `Delete` dari @nestjs/common
    2. Tambah endpoint setelah @Patch(':id'):
       ```
       @Delete(':id')
       @RequirePermissions('school_unit.update')
       @HttpCode(HttpStatus.NO_CONTENT)
       remove(@Param('id', ParseIntPipe) id: number) {
         return this.service.remove(id);
       }
       ```
  </action>
