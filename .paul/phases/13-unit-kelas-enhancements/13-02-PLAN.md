## AC-2: Tab Kelas — tombol Hapus berfungsi
```gherkin
Given tab Kelas aktif, ada data kelas di tabel
When operator klik Hapus → konfirmasi di dialog
Then DELETE /master/classes/:id dipanggil
  AND sukses: list refresh, kelas terhapus
  AND error (ada data terkait): pesan error tampil di dalam dialog
```

## AC-3: Tab Unit Sekolah — Create berfungsi
```gherkin
Given tab Unit Sekolah aktif
When operator klik "Tambah Unit" → isi form (nama, kode) → Simpan
Then POST /master/school-units dipanggil
  AND list unit sekolah direfresh
```

## AC-4: Tab Unit Sekolah — Edit berfungsi
```gherkin
Given tab Unit Sekolah aktif, ada data unit di tabel
When operator klik Edit → ubah data → Simpan
Then PATCH /master/school-units/:id dipanggil
  AND list direfresh
```

## AC-5: Tab Unit Sekolah — Delete berfungsi
```gherkin
Given tab Unit Sekolah aktif, unit sekolah tidak punya kelas terkait
When operator klik Hapus → konfirmasi dialog
Then DELETE /master/school-units/:id dipanggil
  AND sukses: list refresh
  AND error (masih ada kelas): pesan error tampil di dalam dialog
```
