## Backend API (existing, ready)
- `GET /api/billing/payment-templates?classId=N&schoolYearId=N` — list dengan JOINED className, paymentPostName, schoolYearName
- `POST /api/billing/payment-templates` — create, body: { classId, paymentPostId, schoolYearId, amount }
- `PATCH /api/billing/payment-templates/:id` — update, hanya field amount
- `GET /api/master/classes` — dropdown kelas
- `GET /api/master/payment-posts` — dropdown POS
- `GET /api/master/school-years` — dropdown tahun ajaran

Note: POST/PATCH return flat columns (no joined names). Setelah create/edit, re-fetch list untuk mendapatkan joined names.

## AC-1: Router & Table
```gherkin
Given user sudah login
When user membuka /master/template
Then tampil header "Template Pembayaran" + tombol "Tambah Template"
  Dan tabel dengan kolom: Kelas, POS, Tahun Ajaran, Nominal, Aksi (Edit)
  Dan nominal diformat Rupiah berwarna hijau
  Dan filter dropdown Tahun Ajaran di header untuk menyaring data
  Dan sidebar "Template Pembayaran" active (highlight hijau)
```

## AC-2: Modal Create/Edit
```gherkin
Given user membuka /master/template
When user klik "Tambah Template"
Then modal dialog tampil dengan 4 field:
  - Kelas (select dropdown dari GET /classes)
  - POS (select dropdown dari GET /payment-posts)
  - Tahun Ajaran (select dropdown dari GET /school-years)
  - Nominal (input number, min 1000)
  Dan semua field required dengan validasi
  Dan saat edit, hanya field Nominal yang bisa diubah (3 FK readonly)
  Dan setelah create/edit berhasil, tabel refresh
```

## AC-3: Unique Constraint Handling
```gherkin
Given kombinasi (Kelas, POS, Tahun Ajaran) sudah ada di database
When user mencoba membuat template dengan kombinasi sama
Then backend return error
  Dan frontend menampilkan error "Template dengan kombinasi tersebut sudah ada" (bukan crash)
```

## AC-4: Loading & Error
```gherkin
Given user membuka /master/template
When data sedang di-fetch
Then skeleton loading 3 baris tampil
  Dan jika API error, banner error merah tampil
  Dan empty state "Belum ada data template" jika daftar kosong
```
