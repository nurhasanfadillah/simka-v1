```gherkin
Given user sudah login

  Dan tombol "Tambah Tahun" membuka modal dialog kosong dengan form validasi

  Dan validasi: semua field required, tahun akhir > tahun mulai
  Dan setelah create/edit berhasil, tabel refresh otomatis
  Dan isActive ditampilkan sebagai badge hijau "Aktif" atau abu-abu "Tidak"
```

## AC-4: Halaman Manajemen Kelas
```gherkin
Given user sudah login
When user membuka /master/kelas
Then tampil header "Manajemen Kelas" + tombol "Tambah Kelas"
```
