## Backend API (existing, ready)
- `GET /api/master/students?school_unit_id=N` — list dengan LEFT JOIN active class
- `POST /api/master/students` — create (nis auto-generated server)
- `PATCH /api/master/students/:id` — update (excludes entryYear)
- `GET /api/master/students/:id/classes` — enrollment history (className, yearName, unitName joined)
- `POST /api/master/student-classes` — enroll ({ studentId, classId, schoolYearId })
- `PATCH /api/master/student-classes/:id/transfer` — transfer kelas ({ classId })
- `GET /api/master/classes` — dropdown untuk enrollment
- `GET /api/master/school-years` — dropdown untuk enrollment
- `GET /api/master/school-units` — dropdown filter
